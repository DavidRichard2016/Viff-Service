Viff Features

Cross Environments

Find page or partial page difference for a specific url between environments (build, UAT, staging, production) in deployment process.

Cross Browsers

Find page or partial page difference cross browsers for a specific url between browsers. (Chrome vs. Firefox)

Integrated Browserstack

Browserstack supply unlimited browsers on real devices for Viff.

Where can you use Viff

Viff is a semi-automatic testing framework integrated with Selenium Webdriver. Because of the selenium's powerful compatibility against multiple browsers, Viff could be used for browsers including browsers on mobile devices (iOS, Android).

Solving your problems for UI style testing

  • Manually and repeatedly test web pages in different environments and browsers, which would take your lots of time.
  • Testing styles depends on Look and Feel, different tester get different results.
  • No automatic style testing , no CSS refactoring. As time goes by, your CSS code will become a mass, which is not maintainable.
