Viff Service
===============

Viff as service in cloud, no need to install again, MATE!!


Dev
---------------

1. ```npm install --g grunt-cli```
2. ```git clone https://github.com/twers/Viff-Service.git```
3. ```npm install```
4. ```grunt dev```
